I'm a hobbyist programmer that enjoys making JavaScript games and Bash scripts on Unix'esque operating systems.

This site was made with one page of HTML, minimal CSS, and a few JavaScript functions. I used a bunch of popovers to simulate multiple pages.

My free time consists of reading books and reviewing them on Instagram.

Disclaimer: I am not a professor nor a wrestler. The name came from a custom character I made in the game Fire Pro Wrestling.

projects

Programs and scripts.

Antlerhorn An HTML5 side-scrolliong shooter game made with the Phaser JavaScript game library. Uses WASD keys to move and spacebar to shoot.

Animaton Example A bare-bones sprite animation example made with Phaser. On web, hold down your mouse in any direction and the player will follow. Works the same on mobile, but with touch controls.

This website Hosted on github pages.

Phaser Game Template A full template for making a game with Phaser. Includes class-based scene functionality.

quickHTML A Bash script that quickly generates HTML, CSS, and JavaScript boilerplate for making static sites (like this one).

weatherFetch A NodeJS script leveraging the Axios library to show the current weather and some system info in the terminal.

bashBooks A pseudo CRUD terminal app written in Bash for keeping track of the books I'm reading.

imageFinder A quick Bash script for finding images in multiple folders and moving them to a single new folder.
